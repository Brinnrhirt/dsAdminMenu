# Player Management
1. General Options
    - Kill
    - Revive
    - Freeze
    - Spectate
    - Go to
    - Bring
    - Sit in vehicle
    - Routingbucket
2. Administration
    - Kick
    - Ban
    - Permissions
3. Extra Options
    - Open Inventory
    - Give Clothing Menu
    - Give An Item
    - Play A Sound
    - Mute Player
4. Player Information List

I deleted the Dealers menu because you need qb-drugs in order to use it
